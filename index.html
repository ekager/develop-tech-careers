Git Some Projects

Alright, learning data structures is great and all, but when are we going to ACTUALLY MAKE SOMETHING. This is how I felt my whole first year of coding. Sorting lists, greedy algorithms, and data structures don't seem like anything software engineers actually do... These things are the foundations that will get you to thinking like a programmer though! But let's get to the actual projects which you will need to have to find any developer job.
You can make anything your heart desires, but here are some examples to start with!

  • Your personal website! Definitely recommended as a must do. Great news is there are SO many tutorials online that you'll have a website up and running in no time. Check out my hints and tricks section for some tips!
  • A mobile app that is either a clone of an app you like or something simple like a notes app.

Make sure whatever projects you're building that you are putting the code on Github or another code sharing platform so that you can link to it on job applications!
